# 📌 Project Overview

AlgoVista AI is a modern full-stack intelligent recommendation system designed to help users choose the most suitable algorithm for a given computational problem.

The system analyzes multiple constraints such as:

* Problem Type
* Input Size
* Memory Constraints
* Recursion Preference
* Graph Density
* Data Characteristics

and generates:

* ✅ Optimal Algorithm Recommendation
* ✅ Time & Space Complexity Analysis
* ✅ Alternative Algorithm Suggestions
* ✅ Real-Time Benchmarking
* ✅ Complexity Visualization
* ✅ Comparative Analysis
* ✅ PDF Report Export

The platform combines deterministic expert-defined rules with a trained Random Forest Machine Learning model to create an explainable and adaptive recommendation engine.

# 🤖 Hybrid Decision Engine

AlgoVista AI uses a two-layer recommendation system:

## Rule-Based Engine

Uses expert-defined deterministic logic for:

* sorting
* searching
* graph selection
* memory-aware recommendations

## Machine Learning Engine

Powered by:

* scikit-learn
* Random Forest Classifier
* synthetic training dataset generation
* confidence-based prediction system

The final recommendation is selected using an ensemble confidence mechanism.

# 🚀 Local Setup

## Backend Setup

```bash
cd backend

python -m venv venv

# Windows
venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run FastAPI server
uvicorn app.main:app --reload
```

Backend runs on:

```bash
http://localhost:8000
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```
